## PlayCN

Clean Playground to customize your own design system on top of ShadCN UI.

Built using Sandpack, React and TailwindCSS.

[Try now](https://playcn.vercel.app/)

https://github.com/user-attachments/assets/8909b6d4-9093-49b1-8a43-0459b7cf5fff

## Features
- Prototype and modify shadcn/ui primitives for your design system
- Light/Dark Mode support
- Switch themes using `Cmd + k` or `Ctrl + k`. Powered by [tweakcn](https://tweakcn.com/)
- Persist changes to primitives across other components

## Setup

```
git clone https://github.com/vcashwin/playcn
cd playcn
pnpm install
pnpm run dev
```
